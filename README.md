# Diagnostic Platform for Cancer Cachexia: Multi-Omics Based Novel Biomarker Discovery

**Authors:** Dae-Hwan Kim, Jebeom Ko  
**Journal:** (manuscript in preparation)

---

## Overview

This repository contains the R analysis scripts used to generate the figures presented in the above manuscript. The study employs an integrative multi-omics approach—encompassing bulk RNA-seq, single-nucleus RNA-seq (snRNA-seq), and LC-MS/MS proteomics—across multiple independent cancer cachexia mouse models and a human patient cohort, to identify and validate circulating biomarkers for cancer cachexia.

---

## Repository Structure

The analysis pipeline is structured chronologically according to the multi-omics workflow:

```text
Serpina3_Cachexia_Repo/
├── 01_Mouse_Bulk_RNAseq/
│   ├── Figure1.Bulkseq_analysis.R
│   └── Figure1_TargetGenes_TPM_Barplot.R
├── 02_Mouse_snRNAseq/
│   └── Figure2.snseq_analysis.R
├── 03_Mouse_Proteomics/
│   ├── Figure3.proteomics_analysis.R
│   └── Figure4.serum_proteomics_analysis.R
├── 04_Human_Bulk_RNAseq/
│   ├── Figure5.patients_bulkRNA_analysis.R
│   ├── Figure5B_IntNMF_euclidean_summary.R
│   └── Figure5B_IntNMF_euclidean_clustering_summary.docx
├── README.md
└── .gitignore
```

### 1. `01_Mouse_Bulk_RNAseq`
- **`Figure1.Bulkseq_analysis.R`**: Integrated bulk RNA-seq meta-analysis across 4 GEO datasets (PCA, Volcano, K-means clustering, GO ORA)
- **`Figure1_TargetGenes_TPM_Barplot.R`**: TPM barplot for key atrophy marker genes (Trim63, Fbxo32, Eda2r)

### 2. `02_Mouse_snRNAseq`
- **`Figure2.snseq_analysis.R`**: Single-nucleus RNA-seq analysis: QC, integration (SCTransform + Harmony), cell type annotation, secretome DEG heatmap

### 3. `03_Mouse_Proteomics`
- **`Figure3.proteomics_analysis.R`**: LC-MS/MS proteomics meta-analysis: Weighted Stouffer volcano, sensitivity analysis, Leave-One-Out (LOO) stability
- **`Figure4.serum_proteomics_analysis.R`**: Mouse serum proteomics volcano plot

### 4. `04_Human_Bulk_RNAseq`
- **`Figure5.patients_bulkRNA_analysis.R`**: Human patient analysis: IntNMF clustering (K=2), DESeq2 (C2 vs C1), GO BP ORA, SERPINA3 correlation plots
- **`Figure5B_IntNMF_euclidean_summary.R`**: IntNMF Euclidean silhouette analysis and DOCX report generation

## Software Requirements

- **R** >= 4.0
- **Bioconductor** packages: `DESeq2`, `Seurat`, `tximport`, `clusterProfiler`, `fgsea`, `org.Mm.eg.db`, `org.Hs.eg.db`, `ComplexHeatmap`
- **CRAN** packages: `ggplot2`, `dplyr`, `tidyr`, `pheatmap`, `IntNMF`, `harmony`, `scDblFinder`, `officer`, `patchwork`, `ggrepel`
- **Alignment**: Kallisto (transcript quantification), biomaRt (transcript-to-gene mapping)

## Usage

Each script is self-contained and can be executed independently. Before running:

1. Set the working directory or modify the `DATA_DIR` / `BASE_DIR` / `OUTPUT_DIR` variables at the top of each script to point to your local data paths.
2. Ensure all required input files (Kallisto output `.tsv` files, metadata `.xlsx`, etc.) are present at the configured paths.
3. Run scripts sequentially where dependencies exist: `Figure5B_IntNMF_euclidean_summary.R` requires objects generated by `Figure5.patients_bulkRNA_analysis.R`.

> **Note:** Raw sequencing data and processed expression matrices are available through the accession numbers listed above. See the Data Availability section of the manuscript for details.
